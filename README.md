# Smart Room Automation using ESP32

## Project Description
This project automates room appliances using an ESP32 microcontroller. It controls lights and fans using Wi-Fi and IoT technology.

## Features
- Light Control
- Fan Control
- ESP32 Based
- Wi-Fi Enabled
- Low Cost

## Components Used
- ESP32
- Relay Module
- Breadboard
- Jumper Wires
- LED
- Fan

## Software Used
- Arduino IDE

## Future Scope
- Voice Control
- Mobile App
- Google Assistant Integration
